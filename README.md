# 🤖 Bot WhatsApp Multi-fonctions

Bot WhatsApp avec **55+ commandes** et **15 jeux** intégralement fonctionnels.

## 🚀 Démarrage

```bash
START.bat
```

## 📁 Fichiers Essentiels

- `index.js` - Bot principal
- `package.json` - Dépendances  
- `questions_data.js` - Questions quiz
- `aouv_questions.js` - Action ou Vérité
- `loup_garou.js` - Jeu Loup-Garou
- `auth/` - Données sauvegardées

## 🎮 Commandes Principales

### Jeux
- `!jeu` - Jeu aléatoire (15 jeux)
- `!aouv` - Action ou Vérité (200+ questions)
- `!loup create/join/start` - Loup-Garou
- `!culture` - QCM Culture
- `!devine` - Devine mot

### Admin
- `!setup` - Configurer bot
- `!warn @user` - Avertir
- `!kick @user` - Expulser
- `!ban @user` - Bannir
- `!tagall` - Mentionner tout le monde

### Utilitaires
- `!help` - Aide complète
- `!scores` - Classement
- `!myid` - Afficher ID
- `!download [URL]` - Télécharger

## 🎯 Configuration

1. Lancez avec `START.bat`
2. Dans un groupe : `!setup`
3. Activez : `!welcome on` et `!antilink on`
