Conseils de securite

        

          

            Evite le spam
            
N'envoie pas trop de messages identiques et evite les commandes en boucle.

          

          

            Pause quand inutile
            
Utilise le bouton Arreter ou le toggle si tu ne veux pas laisser le bot actif.

          

          

            Compte principal
            
Un bot non officiel peut presenter un risque. Le mieux est d'utiliser un numero secondaire.

          

          

            Session privee
            
Ne partage jamais ton dossier auth. Il donne acces a ta session WhatsApp.
